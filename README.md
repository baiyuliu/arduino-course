## 二、 Arduino 课程阶段安排（示例）
### 1. 基础入门（4–6 课时）
- 核心内容：开发环境搭建、基础语法、数字/模拟 I/O、PWM
- 对应实验：Hello Arduino、LED 闪烁/流水灯、电位器控制亮度

### 2. 传感器与执行器（6–8 课时）
- 核心内容：开关/按键、温湿度/光敏/红外/超声波等传感器；直流/步进电机、舵机、蜂鸣器控制
- 对应实验：雨滴/土壤湿度检测、电机调速、舵机角度控制

### 3. 显示与通信（4–6 课时）
- 核心内容：LCD、7 段数码管、MAX7219 点阵、WS2812B；串口、I2C、SPI、蓝牙/BLE、WiFi
- 对应实验：电子时钟、蓝牙调光/钢琴、APP 控制

### 4. 综合项目（6–10 课时）
- 核心内容：多模块整合、库使用、调试优化
- 对应项目：自动浇花、交通灯、红外遥控、RFID 门禁、超声波雷达

## 三、 学完 Arduino 课程可自主制作的作品（日常场景分类）
学完课程后，学生已掌握**数字/模拟 I/O、传感器读取、执行器控制、显示模块、无线通信**五大核心能力，可制作以下贴近生活的实用/创意作品，均复用课程模块技术。

### 1. 智能家居类（最易上手，复用率最高）
#### （1）智能花盆（自动浇花系统）
- 核心技术：土壤湿度传感器 + 直流电机（水泵） + 继电器
- 课程对应实验：30. 土壤湿度传感器实验 + 8. 直流电机实验 + 自动浇花系统综合实验
- 功能：花盆缺水时自动抽水浇水，可搭配 LCD 显示湿度值，或通过蓝牙手机端查看数据

#### （2）智能灯光控制系统
- 核心技术：光敏电阻 + RGB LED + 蓝牙模块 + 舵机（控制窗帘）
- 课程对应实验：20. 光敏电阻实验 + 3. RGB 呼吸灯实验 + 46. 蓝牙模块实验 + 10. 舵机控制实验
- 功能：天黑自动开灯，手机蓝牙调节灯光颜色和亮度，联动舵机自动拉窗帘

#### （3）门窗防盗报警器
- 核心技术：磁簧开关 + 蜂鸣器 + 红外避障传感器 + LED 指示灯
- 课程对应实验：33. 小磁簧开关实验 + 27. 蜂鸣器实验 + 25. 红外避障实验
- 功能：门窗被打开时磁簧开关触发报警，有人靠近门窗时红外传感器也会触发蜂鸣器和闪光灯

### 2. 生活实用工具类（解决日常小问题）
#### （1）桌面温湿度计（带报警功能）
- 核心技术：DHT11/DHT22 温湿度传感器 + 4 位数码管 + 蜂鸣器
- 课程对应实验：23. 温湿度传感器实验 + 16. 4 位数码管实验 + 27. 蜂鸣器实验
- 功能：实时显示桌面温度和湿度，温度过高/过低或湿度过大时自动报警

#### （2）自动感应洗手器
- 核心技术：红外避障传感器 + 直流电机（水泵） + 继电器
- 课程对应实验：25. 红外避障实验 + 8. 直流电机实验
- 功能：手伸到水龙头下方时，红外传感器触发水泵出水，手离开后自动停水，节水又卫生

#### （3）智能台灯（亮度自动调节）
- 核心技术：光敏电阻 + PWM 调节 + 触摸开关
- 课程对应实验：20. 光敏电阻实验 + 5. PWM 调节 LED 亮度实验 + 19. 触摸开关实验
- 功能：环境光暗时自动点亮台灯，触摸开关控制开关，PWM 自动调节台灯亮度，保护视力

### 3. 玩具与创意互动类（趣味性强，适合展示）
#### （1）蓝牙遥控小车
- 核心技术：直流电机 + L298N 电机驱动板 + 蓝牙模块 + 红外避障传感器（可选）
- 课程对应实验：8. 直流电机实验 + 46. 蓝牙模块实验 + 25. 红外避障实验
- 功能：手机蓝牙 APP 控制小车前进、后退、左转、右转，加装红外传感器后可实现自动避障

#### （2）点阵屏表情机
- 核心技术：MAX7219 点阵模块 + 按键模块 + 串口通信
- 课程对应实验：12. MAX7219 点阵实验 + 15. 按键模块实验
- 功能：通过按键切换显示不同的表情（笑脸、哭脸、爱心等），或通过串口接收电脑指令显示自定义图案

#### （3）声控音乐盒
- 核心技术：声音传感器 + 蜂鸣器 + 按键模块
- 课程对应实验：34. 声音传感器实验 + 27. 蜂鸣器实验 + 15. 按键模块实验
- 功能：拍手触发蜂鸣器播放预设音乐，按键切换音乐曲目，声音过大时自动停止播放

### 4. 校园/学习辅助类（适合学生场景）
#### （1）课堂举手提醒器
- 核心技术：按键模块 + LED 指示灯 + 无线射频模块（RF）
- 课程对应实验：15. 按键模块实验 + 2-1. 点亮 LED 实验 + 40. RFID 实验（射频技术复用）
- 功能：学生按下桌面按键，讲台上的对应 LED 灯亮起，老师可快速看到谁要提问，避免课堂混乱

#### （2）自动收笔盒
- 核心技术：红外循迹传感器 + 步进电机 + 限位开关
- 课程对应实验：24. 红外循迹实验 + 9. 步进电机实验 + 18. 限位开关实验
- 功能：将笔放入盒口，红外传感器检测到后，步进电机带动传送带将笔收进盒内，满笔时限位开关触发提示

#### （3）图书馆静音提醒器
- 核心技术：声音传感器 + 震动传感器 + LED 指示灯 + 蜂鸣器（静音版用震动马达）
- 课程对应实验：34. 声音传感器实验 + 44. 震动开关实验 + 2-1. 点亮 LED 实验
- 功能：环境声音超过阈值或桌面震动过大时，LED 灯闪烁提醒，静音版通过震动马达震动提醒，不影响他人

### 5. 进阶空间
若后续学习 STM32/ESP32，可将这些作品升级为**无线联网版**（如 WiFi 远程控制智能花盆）、**高精度版**（如 STM32 控制的云台小车），功能更强大。

## 五、 新加坡主流创客/Arduino 赛事
1.  **IDE Maker（IDE Series）**：面向学生/创客，团队2-3人；提交项目+现场演示；评估创新、技术实现、实用性；5月提交与展示。
2.  **Tan Kah Kee Young Inventors’ Award（TKKYIA）**：学生科创赛事；3月提交，重创新与社会价值；个人/团队均可。
3.  **World Robot Olympiad（WRO）Future Innovators**：STEM 创新赛，团队制；9月提交，聚焦技术创新与应用落地。
4.  **SLINGSHOT**：全球初创赛，有 MVP 可报；多轮筛选，重商业可行性与市场潜力；奖金丰厚+孵化资源。
